## Grid Design

Figma sources:
- Grid web application layout: https://www.figma.com/design/5NrcsNKae03zepuB2Dr1aD/DDS-Core-Tokens?node-id=6047-284&m=dev
- Grid web layout: https://www.figma.com/design/5NrcsNKae03zepuB2Dr1aD/DDS-Core-Tokens?node-id=6092-170&m=dev

### Web application grid
- Layout includes a fixed sidebar/column block at 264px width.
- The main content area contains multiple equal-width columns/cards arranged horizontally in a fluid row.
- Each content item is `flex: 1 0 0` so cards evenly share available width.
- No horizontal gap is applied between the card columns in this view.

### Web grid structure
- Outer content width is 1280px.
- Outer gutters are 32px on the left and right.
- The center content area is flexible and expands to fill the available space.
- Left and right gutters are represented by 32px columns using token colors for visualization.
